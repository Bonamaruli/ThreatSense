# ThreatSense

ThreatSense adalah platform keamanan siber berbasis AI untuk mendeteksi ancaman dari:

- URL Scan
- Email Scan
- File Scan
- Dashboard Monitoring

## Teknologi

### Backend
- FastAPI
- SQLAlchemy
- SQLite

### Frontend
- Next.js
- React
- Tailwind CSS

## Menjalankan Backend

```bash
cd backend
pip install -r requirements.txt
python main.py
```

## Menjalankan Frontend

```bash
cd frontend
npm install
npm run dev
```
